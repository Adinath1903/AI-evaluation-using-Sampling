# Chatbot Error Analysis using Inverse Sampling

## 📌 Overview
This project applies inverse sampling to estimate and compare error probabilities of AI chatbots (Gemini vs Claude). Instead of fixing the sample size, the number of observations is determined based on a fixed number of errors.

## ⚙️ Methods Used
- Inverse Sampling
- Estimation of error probability (p̂)
- Variance and Standard Error calculation
- Confidence Intervals
- Hypothesis Testing (Z-test)

## 📊 Tools & Libraries
- R
- dplyr
- ggplot2

## 📈 Output
- Error probability comparison
- Confidence interval plots
- Sampling process visualization
- Efficiency comparison (number of questions required)

## 📁 Files
- `sampling project.R` → Main analysis script
- `presentation.pptx` → Project explanation

## 🚀 How to Run
1. Open the R script
2. Load dataset (`df`)
3. Run the script
